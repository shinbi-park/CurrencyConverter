# 환율계산기 


간단한 환율계산기입니다

디폴트 환율값은 1,000 KRW -> USD 입니다

금액부분, 환율부분 모두 숫자 입력가능하며 입력한 금액에 따라 금액 -> 환율 순으로 환산됩니다
